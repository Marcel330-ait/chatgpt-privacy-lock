Privacy Policy / 隐私政策

Effective date / 生效日期: 2026-06-25

English

ChatGPT Privacy Lock is a local browser extension that masks selected ChatGPT sidebar areas that may reveal conversation history, pinned chats, projects, library items, or search entry points.

Data handled locally

The extension stores the following data only in chrome.storage.local:

  • Enabled or disabled state
  • Selected language
  • Selected sidebar areas to hide
  • PIN-derived hash and salt
  • Temporary unlock time
  • Failed-attempt cooldown state

Data not collected

ChatGPT Privacy Lock does not collect, transmit, sell, or share personal data.

The extension does not upload ChatGPT conversations, sidebar titles, project names, prompts, files, account details, browsing history, or PIN values to any server.

The extension does not use analytics, tracking pixels, advertising identifiers, or remote code.

Permissions

The extension uses the Chrome storage permission to save local settings and PIN-derived lock data.

Its content script only runs on:

  • https://chatgpt.com/*
  • https://chat.openai.com/*

Security scope

This extension is a shoulder-surfing and casual-access privacy UX layer. It is not a replacement for device lock screen, Chrome profile protection, ChatGPT account security, or enterprise security controls.

中文

ChatGPT Privacy Lock 是一个本地浏览器扩展，用于遮罩 ChatGPT 侧边栏中可能暴露聊天历史、置顶聊天、项目、资料库或搜索入口的导航区域。

本地处理的数据

扩展只会在 chrome.storage.local 中保存以下数据：

  • 开启或关闭状态
  • 语言选择
  • 所选保护区域
  • PIN 的派生哈希值和 salt
  • 临时解锁时间
  • 错误尝试冷却状态

不收集的数据

ChatGPT Privacy Lock 不会收集、传输、出售或共享个人数据。

扩展不会上传 ChatGPT 对话、侧边栏标题、项目名称、提示词、文件、 账号信息、浏览历史或 PIN 到任何服务器。

扩展不使用分析工具、追踪像素、广告标识符或远程代码。

权限

扩展使用 Chrome 的 storage 权限来保存本地设置和 PIN 派生锁定数据。

内容脚本只运行在：

  • https://chatgpt.com/*
  • https://chat.openai.com/*

安全边界

这是用于防肩窥和临时借用电脑场景的隐私 UX 层，不替代设备锁屏、 Chrome 用户配置文件保护、ChatGPT 账号安全或企业级安全控制。
